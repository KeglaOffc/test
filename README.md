# 🎰 Казино Telegram Бот v2.0

Продвинутый бот казино на Python с множеством игр, системой лотереи и рейтингом.

## 🎮 Доступные игры

- **Mines** - Минный тральщик 💣 (+ сканер мин)
- **Monetka** - Орел/решка 🪙 (+ режим x5)
- **Slots** - Игровые автоматы 🎰 (+ двойной режим x3)
- **Dicedarts** - Кубики и дротики 🎲
- **Futandbask** - Футбол/баскетбол ⚽🏀 (5 уровней + пенальти)
- **Sunduk** - Сундук сокровищ 🎁 (NORMAL/EXTREME/MEGA)
- **Lottery** - Лотереи (часовая, мгновенная, Про, пользовательские) 🎫
- **Mining** - Криптомайнинг симулятор ⛏️
- **Qtop** - Рейтинг игроков 🏆 (+ фильтр невидимости)

## 📋 Требования

- Python 3.9+
- Pip (менеджер пакетов Python)

## 🚀 Быстрый старт

### 1. Подготовка окружения

```bash
# Клонируем репозиторий (если на GitHub)
git clone <your-repo-url>
cd казик_отдельный

# Создаем виртуальное окружение
python -m venv venv

# Активируем его
# Windows:
venv\Scripts\activate
# Linux/Mac:
source venv/bin/activate
```

### 2. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 3. Конфигурация

```bash
# Копируем пример конфигурации
cp .env.example .env

# Редактируем .env и добавляем реальный токен:
# TELEGRAM_BOT_TOKEN=ваш_токен_здесь
```

### 4. Запуск бота

```bash
python start.py
```

Или с явным указанием токена:

```bash
# Windows PowerShell:
$env:TELEGRAM_BOT_TOKEN = "ваш_токен"
python start.py

# Linux/Mac:
export TELEGRAM_BOT_TOKEN="ваш_токен"
python start.py
```

## 🔐 Безопасность

⚠️ **НИКОГДА** не публикуйте токен в коде!

### Правильный способ

```bash
# 1. Создайте .env файл
touch .env

# 2. Добавьте в .env:
TELEGRAM_BOT_TOKEN=your_real_token

# 3. Убедитесь что .env в .gitignore
echo ".env" >> .gitignore

# 4. Используйте в коде:
import os
token = os.getenv('TELEGRAM_BOT_TOKEN')
```

## 🐛 Решение проблем

### Проблема: "Модуль не найден"

```bash
# Убедитесь что зависимости установлены:
pip install -r requirements.txt

# Если проблема в импорте Handlers:
python -c "from Handlers import admin"  # Проверяем импорт
```

### Проблема: "Токен невалиден"

```bash
# 1. Проверьте что токен правильно установлен:
echo %TELEGRAM_BOT_TOKEN%  # Windows
echo $TELEGRAM_BOT_TOKEN   # Linux

# 2. Убедитесь что это не старый токен
# 3. Получите новый у @BotFather
```

### Проблема: Бот не отвечает

```bash
# 1. Проверяем логи
tail -f bot.log

# 2. Проверяем интернет соединение
ping api.telegram.org

# 3. Убедитесь что бот не запущен несколько раз
# ps aux | grep python  # Linux
# Get-Process python    # Windows PowerShell
```
